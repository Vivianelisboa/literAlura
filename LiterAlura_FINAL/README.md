
# 📚 LiterAlura - Projeto Final

Projeto completo utilizando:

- Java 17
- Spring Boot 3.2.3
- PostgreSQL
- Spring Data JPA
- Jackson 2.16

## Funcionalidades

✔ Listar livros  
✔ Listar autores  
✔ Autores vivos por ano  
✔ Contar livros por idioma  
✔ Persistência automática  

## Como executar

1. Criar banco:
   CREATE DATABASE literalura;

2. Ajustar usuário e senha

3. Executar:
   mvn spring-boot:run
